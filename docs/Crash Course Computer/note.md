# 四-二进制

- 二进制中，一个1或者0叫做“位”

- “8位”是很常见的单位，被称为“字节”   B  就是字节

   - 大部分计算机当中，第一位都表示“正负号”

- 计算机中记录内存位置的标记   “位址”

   - 位址通常也有64位
